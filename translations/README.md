# Translations

Welcome to MakerDAO Translation Initiative. Our common goal is to bring MakerDAO educational content to people around the world in their native languages. 
This is a community-lead initiative that uses bounties to incentivize translations of various resouces. Below are some of the resources that have been created as a result of this initiative.
